### **user (사용자 기본 정보)**

| 속성               | 데이터 타입        | 필수 여부 | 설명                              | 예시                     |
| :----------------- | :----------------- | :-------- | :-------------------------------- | :----------------------- |
| **userId (PK)**    | INT                | 필수      | 사용자 고유 ID                    | `1`                      |
| **email**          | VARCHAR(30)        | 필수      | 사용자 이메일                     | `"test@example.com"`     |
| **password**       | VARCHAR(30)        | 필수      | 사용자 비밀번호                   | `"p@ssw0rd!"`            |
| **name**           | VARCHAR(20)        | 필수      | 사용자 이름                       | `"홍길동"`               |
| **birthDate**      | DATE               | 필수      | 사용자 생년월일 (YYYY-MM-DD)      | `"2010-05-18"`           |
| **phoneNumber**    | VARCHAR(30)        | 필수      | 사용자 휴대폰 번호                | `"010-1234-5678"`        |
| **nickname**       | VARCHAR(30)        | 필수      | 사용자 닉네임                     | `"mathKing"`             |
| **gender**         | VARCHAR(30) (ENUM) | 필수      | 성별 정보 (`'male'`, `'female'`)  | `"male"`                 |
| **emailVerified**  | BOOLEAN            | 필수      | 이메일 인증 여부                  | `FALSE`                  |
| **createdAt**      | DATETIME           | 필수      | 회원가입 완료 시각 (UTC ISO 포맷) | `"2025-07-10T12:00:00Z"` |
| **isActive**       | BOOLEAN            | 필수      | 사용자 활성화 상태                | `TRUE`                   |
| **agreeTerms**     | BOOLEAN            | 필수      | 서비스 이용약관 동의 여부         | `TRUE`                   |
| **agreePrivacy**   | BOOLEAN            | 필수      | 개인정보 수집·이용 동의 여부      | `TRUE`                   |
| **agreeMarketing** | BOOLEAN            | 선택      | 마케팅 정보 수신 동의 여부        | `FALSE`                  |

| 속성                | 데이터 타입 | 필수 여부   | 설명                          | 예시              |

| **phoneNumber**     | VARCHAR(30) | 조건부 필수 | 법정대리인 연락처 (14세 미만) | `"010-1234-5678"` |
| **consent**         | BOOLEAN     | 조건부 필수 | 법정대리인 동의 여부          | `TRUE`            |
